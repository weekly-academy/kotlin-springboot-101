## 주요 기능 사항 정리

### WORDLE 게임 조건
* 하루 한번만 문제를 풀 수 있는 기능
* 5자리 문자를 입력 받는다
  * 작거나 크면 입력 불가
* 사전에 있는 단어만 입력 가능
  * 사전에 없다면 입력 불가 
* 게임 할 수 있는 기회는 6회(그 이상 진행 할 수 없다)
  * 5자리 모두 6회 전에 맞추었을 경우 게임 종료
  * 맞추지 못 했을 경우 해당 정답 제시하는 기능 

### WORDLE 게임
* 입력한 5자리 문자와 정답인 5자리 문자를 비교
  * 위치와 문자 모두 같다면 녹색
  * 위치는 다르지만 해당 문자를 포함한다면 노란색
  * 정답 문자에 없는 경우 회색

### WORDLE 게임 결과
* 결과를 공유 할 수 있는 기능
  * SNS를 통한 공유
  * 게임 결과를 공유 할 수 있는 기능
